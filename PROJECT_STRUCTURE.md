## File Counts

- **Total Pages**: 11 static pages + 3 dynamic routes
- **Example Content**: 3 projects, 2 articles, 4 notes, 2 research papers, 1 newsletter
- **Layout Files**: 1 base layout
- **Configuration**: 5 config files

## Key Features by File

### Core Layout
- `BaseLayout.astro`: Fixed sidebar navigation, mobile menu, SEO meta tags, responsive design

### Homepage (`index.astro`)
- Hero section with positioning statement
- Selected projects (top 3)
- Latest articles (top 3)
- Notes grid (top 4)
- Research highlights (top 2)
- Newsletter CTA

### Content Pages
- **About**: Bio, work focus, problems, beyond work sections
- **Projects**: Industry-problem-approach-impact structure
- **Articles**: Long-form essays with tags and full typography
- **Notes**: Brief thoughts in grid layout
- **Research**: Papers with institutional context
- **Newsletter**: Subscription forms for multiple newsletters

### Technical Features
- Content collections with TypeScript schemas
- RSS feed for articles
- Sitemap (auto-generated by Astro)
- SEO optimization (meta tags, Open Graph, Twitter cards)
- Mobile-responsive design
- Dark theme with custom color system
